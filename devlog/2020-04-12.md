**How to represent Uci.Position?**
* ❌Position [Move]

* ❌String -> Move
    * 😡Handling errors is impossible with this
* ❌String -> Maybe Move
    * 🙂Simplest way
    * 😡No error context 
* ✅String -> Either String Move
    * 🙂Error context
